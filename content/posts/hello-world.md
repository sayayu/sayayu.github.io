## Markdownで記事を書く

記事はすべて `content/posts/` 配下の Markdown ファイルで管理します。新しい記事を作るには以下のコマンドを実行してください。

```
hugo new content posts/my-new-post.md
```

見出し、リスト、コードブロック、画像など、通常の Markdown 記法がそのまま使えます。

- シンプル
- 読みやすい
- 高速

> Markdown だけで更新できるので、執筆に集中できます。
